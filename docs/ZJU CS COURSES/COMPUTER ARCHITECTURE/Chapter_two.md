# Chapter Two: Memory Hierarchy Design

## 2.1 Introduction

### SRAM Technology

* SRAMs don’t need to refresh and so the access time is very close to the cycle
time. SRAMs typically use six to eight transistors per bit to prevent the information from being disturbed when read. SRAM needs only minimal power to retain the
charge in standby mode.
* Nowadays, SRAM are integrated into the processor chip.
